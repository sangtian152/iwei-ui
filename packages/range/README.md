# zmbl-range
> A zmbl component

## Installation
```shell
npm i zmbl-range -D
```

## Usage
```javascript
import Vue from 'vue'
import MintRange from 'zmbl-range'

Vue.use(MintRange)
```

or

```javascript
import Vue from 'vue'
import { MintRange } from 'zmbl-range'

Vue.component('zmbl-range', MintRange)
```

## Development

```shell
make dev
```

## Production
```
make dist
```

## Deploy
```shell
make deploy
```
